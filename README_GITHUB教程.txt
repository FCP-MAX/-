
# PrintColorPro Android 项目

## ✅ 用手机直接在 GitHub 打包 APK步骤

### 1. 打开 GitHub 创建仓库
- New repository → 输入名称 → Create

### 2. 上传本压缩包内容到仓库
- Upload files → 选择所有文件 → Commit

### 3. 打开 Actions
- 点击 `Actions`
- 找到 `Android Build`
- 点击 `Run workflow`

### 4. 等待构建完成
- 等待 2~5 分钟
- 在页面底部找到 `Artifacts`
- 下载 `app-debug.apk`

### 5. 在安卓手机安装
- 允许未知来源
- 安装 apk
